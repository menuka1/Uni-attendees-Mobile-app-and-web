200 : The request was successful.
201 : Responding back to the request.
400 : The request had bad syntax or was inherently impossible to be satisfied. (Argument not found)
401 : Authentication failed, probably because of invalid/missing API key. (Forbidden)
402 : The usage limit has been reached, either you exceeded per day requests limits or your balance is insufficient.
403 : You are not authorized to perform this operation. (Unauthorized)
404 : The requested resource was not found. (Not found / Unsuccessful request)
405 : The requested method was not found.
409 : The resource already exists.
500 : Ops. Something were wrong. (Server side error!)
503 : Our system is a bit busy at the moment and your request can’t be satisfied.
